# ♠️ Python Poker Game vs Computer

A terminal-based Poker game built using Python where the player competes against an AI opponent. The project focuses on Object-Oriented Programming (OOP), game logic, card handling, and basic AI decision-making.

## 🚀 Features

* Play Poker against the computer.
* Automatic deck creation and shuffling.
* Card dealing system.
* Hand evaluation system:

  * High Card
  * Pair
  * Two Pair
  * Three of a Kind
  * Straight
  * Flush
  * Full House
  * Four of a Kind
  * Straight Flush
  * Royal Flush
* Betting system with virtual chips.
* Computer AI that makes decisions based on hand strength.
* Multiple rounds until one player loses all chips.
* Scoreboard and statistics.

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (Classes & Objects)
* Random Module
* File Handling (for saving progress and statistics)
